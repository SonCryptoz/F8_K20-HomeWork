Semantic HTML là gì?

Semantic HTML là cách sử dụng các thẻ HTML mang ý nghĩa rõ ràng như <header>, <main>, <section>, <article> hoặc <footer> để mô tả cấu trúc nội dung website.

Thay vì sử dụng quá nhiều <div>, semantic HTML giúp trình duyệt, search engine và screen reader hiểu rõ vai trò của từng phần nội dung.

Trong phát triển website hiện đại, semantic HTML gần như là tiêu chuẩn bắt buộc nếu muốn xây dựng website chuyên nghiệp và dễ maintain.

Một điểm mình rất thích là semantic HTML giúp code dễ đọc hơn rất nhiều. Khi nhìn vào cấu trúc HTML, developer khác có thể nhanh chóng hiểu layout của website.

Ngoài ra, semantic HTML còn hỗ trợ SEO và accessibility cực kỳ tốt .

SEO và Accessibility

Search engine như Google sử dụng semantic structure để hiểu nội dung chính của website.
Ví dụ: <article> thường chứa nội dung bài viết, còn <nav> dùng cho navigation menu.

Accessibility cũng là yếu tố rất quan trọng. Những người sử dụng screen reader sẽ dễ dàng điều hướng website hơn khi cấu trúc semantic được xây dựng chính xác.

Khi code, mình thường dùng phím tắt Ctrl + S để lưu file nhanh và kiểm tra lại semantic structure thường xuyên.

Nếu cấu trúc đúng, trình validator có thể trả về output như: Document checking completed. No errors or warnings found.

Ví dụ Semantic HTML

Đây là ví dụ về một layout semantic cơ bản: 

<!-- Header của website -->
<header class="header">
    <nav class="navbar">
        <ul class="nav-list">
            <li>
                <a href="/">Trang chủ</a>
            </li>
        </ul>
    </nav>
</header>

<!-- Nội dung chính -->
<main>
    <section class="hero">
        <h1>Semantic HTML</h1>
    </section>
</main>

<!-- Footer website -->
<footer>
    <p>Copyright 2026</p>
</footer>

Việc chia layout như trên giúp website rõ ràng hơn thay vì chỉ dùng nhiều <div>.

Ví dụ CSS Layout

Dưới đây là ví dụ CSS đơn giản sử dụng Flexbox: 

/* Container layout */
.container {
    display: flex;
    align-items: center;
    justify-content: space-between;
    gap: 24px;
}

/* Responsive mobile */
@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}

Mình thường kết hợp semantic HTML với Flexbox và CSS Grid để xây dựng layout responsive hiện đại.
